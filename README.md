# AI & IoT-Based Semi-Autonomous Firefighting Drone

## Overview
This project focuses on developing a semi-autonomous UAV system for fire monitoring and emergency response applications using IoT technologies and embedded systems.

## Features
- UAV-based firefighting concept
- IoT sensor integration
- Flight controller integration
- Embedded system coordination
- Autonomous operational workflow

## Technologies Used
- Arduino
- KK2
- Brushless Motors
- ESCs
- IoT Sensors

## My Contribution
- Worked on embedded system integration
- Coordinated sensor-based workflows
- Assisted in UAV assembly and system functionality
- Led a team of 5 members during project execution
